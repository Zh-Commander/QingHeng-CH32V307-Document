# MounRiver Studio 帮助手册

V1.30

## 一、概述

MounRiver Studio 是一款针对嵌入式开发的集成开发环境。MounRiver Studio 提供了包括 C 编译器、宏汇编、链接器、库管理、强大的调试器和下载器等在内的完整开发方案，通过一个集成开发环境将这些部分组合在一起。

## 二、计算机配置要求

### 2.1 最低配置

CPU 主频： 1 GHz

内存： 2G

硬盘剩余空间： 1GB

操作系统： Windows7

### 2.2 推荐配置

CPU 主频： 2 GHz 及以上

内存： 2GB 及以上

硬盘剩余空间： 2GB 及以上

操作系统： Windows 7 及以上
