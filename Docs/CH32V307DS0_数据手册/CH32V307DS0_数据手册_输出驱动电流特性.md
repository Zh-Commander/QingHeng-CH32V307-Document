# 输出驱动电流特性

GPIO（通用输入/输出端口）可以吸收或输出多达±8mA电流，并且吸收或输出±20mA电流（不严格达到 $V _ { \mathfrak { a } } / \mathsf { V } _ { \mathfrak { d } \sharp } ,$ ）。在用户应用中，所有IO引脚驱动总电流不能超过4.2节给出的绝对最大额定值：
