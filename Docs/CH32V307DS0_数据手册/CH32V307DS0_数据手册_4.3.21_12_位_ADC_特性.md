公式：最大 $\mathsf { R } _ { \mathsf { A l N } }$

$$
\mathrm {R _ {A I N} <   \frac {T _ {S}}{f _ {A D C} \times C _ {A D C} \times \ln 2 ^ {N + 2}} - R _ {A D C}}
$$

上述公式用于决定最大的外部阻抗，使得误差可以小于1/4 LSB。其中 $N = 1 2$ （表示12位分辨率）。
